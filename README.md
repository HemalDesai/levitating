### Installation

Follow these steps to setup the website.


1. Clone the repo
   ```sh
   git clone https://github.com/HemalDesai/levitating.git
   ```
2. Navigate to the directory
   ```sh
   cd levitating
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the server
    ```sh
    npm run dev
    ```
